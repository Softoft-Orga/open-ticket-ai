# Changelog

All notable changes to the Open Ticket AI OTOBO/Znuny Plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release as standalone PyPI package
- OTOBO/Znuny/OTRS ticket system integration
- Support for ticket search and retrieval
- Support for ticket updates (queue, priority, etc.)
- Support for adding notes/articles to tickets
- Unified ticket model for seamless Open Ticket AI integration
- Configuration via YAML with environment variable substitution
- Comprehensive documentation and usage examples

### Changed
- Extracted from main Open Ticket AI repository into standalone package
- Package now available on PyPI as `open-ticket-ai-otobo-znuny-plugin`
