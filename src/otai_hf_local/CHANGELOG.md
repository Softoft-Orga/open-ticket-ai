# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release as standalone PyPI package
- HFLocalTextClassificationPipe for local Hugging Face model inference
- Support for sequence classification models
- Model caching functionality
- Jinja2 template support for prompts
- Comprehensive test suite

### Features
- Local execution without external API dependencies
- Configurable model and token parameters
- Confidence score output alongside predictions
