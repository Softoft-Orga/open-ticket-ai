# open-ticket-ai-hf-local

HuggingFace local inference plugin for Open Ticket AI.

This plugin provides local text classification capabilities using HuggingFace Transformers models.

## Installation

```bash
pip install open-ticket-ai-hf-local
```

## Requirements

- Python >= 3.13
- open-ticket-ai >= 1.0.0rc1
- transformers[torch] ~= 4.52.4

## Usage

This plugin integrates with the Open Ticket AI pipeline system. See the main [Open Ticket AI documentation](https://open-ticket-ai.com) for configuration examples.
