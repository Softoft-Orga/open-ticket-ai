# Test Data

Dieses Verzeichnis enthält Testdaten für die verschiedenen Testsuiten.

## Richtlinien

- Testdaten sollten klein und textbasiert sein (YAML/JSON bevorzugt)
- Keine großen Binärdateien
- Keine realen oder sensiblen Daten
- Strukturiere Daten nach Testtyp in Unterverzeichnissen bei Bedarf
