# Getting Started with Open Ticket AI

This guide will help you set up and configure Open Ticket AI for your ticket system.

## Prerequisites

Before you begin, ensure you have the following:

- Python 3.13 or higher
- Access to your ticket system (OTOBO, Znuny, Zammad, etc.)
- Basic understanding of YAML configuration

## Installation

Install Open Ticket AI using uv:

```bash
uv pip install open-ticket-ai
```
