## Design System Principles

The Open Ticket AI design system follows these core principles:

- **Color Palette:** Deep purple/cyan glow with dark surfaces
- **Typography:** Inter font family with consistent sizing
- **Spacing:** Generous spacing for clarity
- **Accessibility:** WCAG 2.1 AA compliant
- **Responsiveness:** Mobile-first approach

## How to Use This Documentation

Navigate through the sidebar to explore individual components. Each component section above includes:

- **Interactive examples** with live previews
- **Code snippets** showing how to use the component
- **Variants** demonstrating different states and configurations

Click on any story to see its detailed documentation, including props, controls, and additional examples.
