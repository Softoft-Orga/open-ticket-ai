# Open Ticket AI

Open Ticket AI is an open-source, on-premise solution that automatically classifies support tickets by queue and priority. It integrates seamlessly with OTOBO, Znuny, and OTRS ticket systems.

## Key Features

- **Automatic Classification**: AI-powered queue and priority assignment
- **On-Premise Deployment**: Keep your data secure within your infrastructure
- **Plugin Architecture**: Extensible system with support for custom plugins
- **Pipeline-Based Processing**: Flexible workflow orchestration
- **Template System**: Dynamic configuration using Jinja2 templates
- **Dependency Injection**: Clean, testable architecture
